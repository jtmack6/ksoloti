Ksoloti Patcher Keyboard Shortcuts

Note: Key combos are written in all capitals but you should only press SHIFT when it is explicitly shown in the combo.

On MacOS use the CMD key wherever the CTRL key is used in this document.

Patch Window: General

Key Combo

Function

Notes
CTRL+N New Patch Creates an empty new patch window.
CTRL+O Open... Opens a file selector to open an existing patch file. The path will default to wherever you last opened or saved a patch using this menu.
CTRL+S Save Saves the changes made to the patch.
CTRL+SHIFT+S Save As... Opens a file selection dialog to save the patch in its current state as a new file.
CTRL+W Close Closes the current patch window.
CTRL+E Toggle Live Mode Takes your patch live or brings it back to edit mode.
CTRL+X Cut Cuts the selected objects, trying to keep their connections intact, and places them in the clipboard.
CTRL+C Copy Copies the selected objects to the clipboard, trying to keep their connections intact.
CTRL+V Paste Pastes the content of the clipboard.
CTRL+D Duplicate Creates a copy of the selected objects trying to keep their connections intact, pastes them directly to the canvas and activates mouse drag.
Left-click to confirm placement of the objects.
A copy of the duplicated content will remain in the clipboard so you can use the paste action to create more duplicates. This is essentially just a shortcut for the three actions [copy, paste, activate mouse drag].
DEL Delete Deletes the currently selected objects and related connections.
CTRL+A Select All Selects all objects in the current patch. Useful if you want to copy everything to a subpatch, or realign all objects in your patch window.
CTRL+Z Undo Reverts the last operation. (Currently not all operations are covered, e.g. changing comment text or object instance names has no undo.)
CTRL+Y, CTRL+SHIFT+Z Redo Re-does the last operation. In other words, it reverts the most recent undo. (Currently not all operations are covered, e.g. changing comment text or object instance names has no undo.)
CTRL+I Open Patch Settings Opens the current patch's Settings window, where settings like the MIDI channel it listens on, subpatch mode (including polyphony) and other options can be changed.
CTRL+T Open Patch Notes Opens the current patch's Notes where you can save free notes and comments about your patch.
CTRL+B Toggle Patch Cords in Background Toggles patch cord visibility either on top or hidden by the objects. Convenient in complex patches or when you're done editing and want to have a cleaner canvas.
SHIFT Snap to Grid While dragging objects, hold down SHIFT to disable the snap to grid function.
SHIFT, CTRL+SHIFT Fine Tune While adjusting a parameter or attribute, hold down SHIFT to fine-tune it.
Hold down CTRL+SHIFT to fine-tune it in the smallest steps possible.
TAB, SHIFT+TAB Cycle Focus Cycles through the available attributes or parameters in yoru patch.
CTRL+SHIFT+M Show Main Window Takes you to the main window. Useful if you want to jump there to check the console status or quick-connect/disconnect your board.
CTRL+SHIFT+Y Show Virtual Keyboard Takes you to the virtual keyboard window. The virtual keyboard simulates MIDI events as if they were received via DIN or USB. You can use it to test your patch.
CTRL+SHIFT+F Show File Manager Takes you to the file manager window. Here you can view, add, and delete files on your SD card.
SPACE Show Object Finder Opens the Object Finder. Here you can search for and add objects to your patch. The search field contains your last entered search string.

Patch Window: Object Finder Shortcuts

In the patch window, press the corresponding key combo to open the Object Finder with the string already pasted in the search field.

You can find four additional slots for your own custom shortcuts in the preferences. Trigger these using SHIFT+the "1" to "4" keys after you've set them. Note that the Object Finder (including these shortcuts) also supports the "*" wildcard and Regex.
